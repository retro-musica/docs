## 投稿手順
### 記事は `/items/<記事名>/index.md` に記述

- 全ての記事は `/items/<記事名>/index.md` に記述しましょう。場所を決めておくことで共同編集がしやすくなります。
- 該当年月ディレクトリの `index.md` に記事のリンクを追記
- 必ずどのファイルにも `[ホームへ戻る](/home)` を記述
